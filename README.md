### 📋제출 방법

1. 본 레포지토리를 fork한다.
2. fork한 레포지토리를 clone한다.
3. 기능 개발을 시작한다.
4. 각 기능이 개발될 때마다 commit을 한다.
5. 프로그램 개발이 완료되면 PR을 제출한다.

   - PR 제목은 **4회차 과제-이름**으로 작성해 주세요.

### PR에 포함되어야 할 내용

1. 내가 개발한 기능
2. 내가 개발할 때 유의깊게 개발한 부분
3. 내가 개발하면서 들었던 의문 사항
